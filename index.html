The Basics

HTML is not a programming language, it's a markup language. It tells the browser what to do. HTML isn't really that hard, in fact, ChatGPT says it's one of if not the easiest! Let's get it down

So, let's get over tags. Tags are CRUCIAL for making the site work, and they serve the purpose of helping elementsdictate what should be on the HTML file.

Here are the top 10 main tags and their meanings!

  1. h1...h6 - Headers from 1 to 6, 1 being the biggest.
  2. section - Tag that sets things apart.
  3. p - Paragraph (not an actual one, but a wall of words unless you close the tag)
  4. i - Italicizes a word (e.g. Hello!)
  5. b - Makes a word bold (e.g. Hi there!
  6. u - Underlines a word (e.g. What's up?
  7. !DOCTYPE html - Tells the site that it's an HTML5 site
  8. html - Wraps the code content.
  9. head - Basically the brain of the code of the site.
  10. title - The text in the browser tab.

Indents

Indents are useful in the code for spacing certain elements out, we'll get to elements later. They are extraodinarily helpful for cleaning up jumbled code and giving you the freedom to get a shield from headaches!

Elements

Elements are a line or wall (depends on the text style) of bits of the HTML code that tell the browser what it is and show it to the user.
